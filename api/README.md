An API application to keep track of habits, to-dos, dailies, notes, blog post, pro-con list.

# Technology

- Python: v3.10.5
- pyenv
- pipenv
- [Flask](https://flask.palletsprojects.com/en/2.1.x/)

## Install Dependencies

```
pipenv install
```

## Environment

Activate a virtual environment.

```
pipenv shell
```

## Run

```
flask run
```
